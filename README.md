# Adhikaar

**A verifiable public-benefit reasoning engine.** Describe your situation in plain language; Adhikaar tells you which Indian central government welfare schemes you're eligible for, **why**, what documents you need, and how to apply — with every eligibility claim grounded in and cited to official scheme text.

> Core principle: **it never invents an entitlement.** Eligibility is decided by a deterministic, unit-tested rules engine whose every rule links to its source clause. The LLM only structures your input and explains the verdict — it never decides.
